Smart Campus Management System
A full-stack Smart Campus Management System built using C++, React, TypeScript, and SQLite to simplify campus operations through role-based dashboards for students, teachers, and admins.

Features:
Role-Based Authentication
Course Management
Attendance Tracking
Assignment Management
Grade Management
REST API Integration

Tech Stack
Frontend: React, TypeScript, HTML, CSS
Backend: C++, REST APIs
Database: SQLite

Architecture:
Frontend (React)
       ↓
REST APIs
       ↓
C++ Backend
       ↓
SQLite Database
